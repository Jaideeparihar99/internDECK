Full‑stack app • Backend: Express • Frontend: React (CRA)

CampusConnect

This repository contains a Node/Express backend at the repo root and a React frontend inside frontend/. If you opened this file directly, you’ll only see this landing page—start the dev servers to run the app.

Start backend (API)

From the repo root:
npm install npm run dev

Start frontend (React)

In frontend/:
cd frontend npm install npm start

Notes

The real React HTML template is frontend/public/index.html. The frontend dev server will open the app in your browser automatically.
